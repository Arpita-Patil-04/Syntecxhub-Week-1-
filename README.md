# Syntecxhub-Week-1-
A Python-based data preprocessing tool built using pandas that automates common data cleaning tasks such as handling missing values, correcting data types, removing duplicates, and standardizing column names. This project is designed to streamline raw dataset preparation for analysis and machine learning workflows.

Features
🔹 Detects and handles missing values
Numeric → filled with median
Categorical → filled with mode
🔹 Converts incorrect data types
String → Numeric
String → Datetime
🔹 Standardizes column names
Lowercase, removes spaces, replaces with underscores
🔹 Removes duplicate records
🔹 Generates a cleaned dataset
🔹 Creates a cleaning log file for transparency

Tech Stack
Python
pandas
Google Colab / Jupyter Notebook
